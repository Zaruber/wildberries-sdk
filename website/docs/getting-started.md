# Начало работы

Получите API-токен в личном кабинете продавца Wildberries и передайте его клиенту.
Ниже — установка для каждого языка.

=== "Python"

    ```bash
    pip install wildberries-sdk
    ```

=== "Node.js"

    ```bash
    npm install wildberries-sdk
    ```

=== "Go"

    ```bash
    go get github.com/eslazarev/wildberries-sdk/clients/go/finances
    ```

=== "PHP"

    ```bash
    composer require eslazarev/wildberries-sdk
    ```

=== "Rust"

    ```bash
    cargo add wildberries_sdk_finances
    ```

Конкретные примеры вызовов смотрите на страницах операций в разделе «Справочник API».
